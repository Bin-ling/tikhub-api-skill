## 技能列表

### SK-MON-002: 全赛道带货转化极致优化
**适用场景**: 电商带货、团购、小程序转化
**核心目标**: 带货转化率≥5%
**输入**: 选品、脚本、人群、商品信息
**微步骤**:
1. 选品严格匹配人群需求
2. 逻辑：痛点→产品→效果→信任
3. 前5秒出现产品主体
4. 口播只讲核心利益点
5. 评论区+小黄车双重引导
**输出**: 带货转化优化方案
**顶级量化判断**: 转化率≥5%
**AI执行规则**: 基于真实价值，禁止夸大宣传

### SK-MON-003: 全赛道商业变现模式设计
**适用场景**: 账号商业化、收益多元化
**核心目标**: 变现渠道≥5个，月收入增长≥50%
**输入**: 账号定位、粉丝画像、行业趋势
**微步骤**:
1. 变现模式评估与选择
2. 多元化变现渠道设计
3. 变现路径优化
4. 粉丝价值挖掘
5. 收益目标与增长策略
**输出**: 商业变现模式设计方案
**顶级量化判断**: 变现渠道≥5个，月收入增长≥50%
**AI执行规则**: 变现设计必须基于粉丝需求，避免过度营销
