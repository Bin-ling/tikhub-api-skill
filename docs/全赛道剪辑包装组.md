## 技能列表

### SK-EDI-001: 全赛道0.5s节奏卡点剪辑
**适用场景**: 所有短视频成片、节奏优化
**核心目标**: 节奏紧凑，3秒留人率≥75%
**输入**: 拍摄素材、脚本、BGM
**微步骤**:
1. 对齐BGM鼓点，0.5-1s切镜
2. 删除所有停顿、喘气、废话
3. 高潮前置：最精彩画面放0-3s
4. 干货快剪，情绪慢放
5. 结尾留帧2s做关注引导
**输出**: 高节奏成片
**顶级量化判断**: 3秒留存率≥75%
**AI执行规则**: 无空镜、无废话、无拖沓

### SK-EDI-002: 全赛道标准化字幕制作
**适用场景**: 全平台视频字幕、信息强化
**核心目标**: 关键信息一眼识别，抓取率≥98%
**输入**: 脚本、成片视频
**微步骤**:
1. 字体：粗体无衬线
2. 大小：≥36号（9:16）
3. 颜色：黑底白字/白底黑字
4. 关键词高亮：红/黄色标色
5. 每行≤7字，居中或左下布局
**输出**: 标准化字幕成片
**顶级量化判断**: 信息抓取率≥98%
**AI执行规则**: 核心关键词必须高亮
