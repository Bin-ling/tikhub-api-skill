## 技能列表

### SK-SHO-006: 全赛道标准化灯光布控
**适用场景**: 真人出镜、画质质感统一
**核心目标**: 画面通透、无阴影、肤色自然
**输入**: 场景、环境光、人物位置
**微步骤**:
1. 主光：45°角，亮度最高
2. 辅光：亮度为主光50%，弱化阴影
3. 轮廓光：分离人物与背景
4. 色温固定5600K自然光
5. 禁止顶光、逆光、阴阳脸
**输出**: 灯光布控图
**顶级量化判断**: 画面无阴影，肤色通透
**AI执行规则**: 优先保证人脸清晰明亮

### SK-SHO-007: 全赛道收音质量极致控制
**适用场景**: 口播、剧情、所有有声视频
**核心目标**: 人声100%清晰，杂音率=0
**输入**: 录音设备、拍摄环境
**微步骤**:
1. 领夹麦距嘴5-10cm
2. 关闭风扇/空调/窗外环境噪音
3. 加装防风棉防喷麦
4. 音量控制-12dB，无爆音
5. 每段录音即时回听校验
**输出**: 高清无噪干音
**顶级量化判断**: 杂音率=0，人声清晰度100%
**AI执行规则**: 任何杂音必须重录
