## 技能列表

### SK-SHO-001: 全赛道景别精准选择
**适用场景**: 分镜脚本、拍摄指挥、镜头规划
**核心目标**: 镜头语言100%服务内容，无多余镜头
**输入**: 脚本台词、动作、情绪、内容重点
**微步骤**:
1. 特写：细节/表情/产品/文字（0.5-1s）
2. 近景：上半身/口播/动作（1-2s）
3. 中景：全身动作/互动（2-3s）
4. 全景：环境/氛围/场景（1-2s）
5. 空镜：转场/过渡/氛围（0.5s）
6. 优先级：特写>近景>中景>全景
**输出**: 分镜景别执行表
**顶级量化判断**: 景别匹配度100%，无长镜头
**AI执行规则**: 单镜头时长≤3s，禁止拖沓

### SK-SHO-002: 全赛道运镜动作执行
**适用场景**: 拍摄运镜、节奏强化、画面流畅
**核心目标**: 运镜服务节奏，不炫技、不突兀
**输入**: 分镜、脚本、BGM节奏
**微步骤**:
1. 固定：口播/讲解（稳定无抖动）
2. 推：强调重点/产品/细节
3. 拉：展示场景/环境
4. 摇：展示细节/流程
5. 跟：跟随人物/动作过程
6. 甩：快节奏/转场/高能点
**输出**: 运镜执行方案
**顶级量化判断**: 运镜与节奏匹配度100%
**AI执行规则**: 运镜必须有明确目的
