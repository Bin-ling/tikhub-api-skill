## 技能列表

### SK-OPR-001: 全赛道精准发布时间匹配
**适用场景**: 全平台发布、初始流量最大化
**核心目标**: 初始流量池≥1.2万播放
**输入**: 人群画像、平台活跃数据
**微步骤**:
1. 学生党：12:00/17:00/21:00
2. 职场人：08:00/12:00/19:00
3. 宝妈：10:00/15:00/21:30
4. 测试3个时段，固定最优
5. 发布前10分钟预热评论区
**输出**: 账号专属发布时间表
**顶级量化判断**: 初始流量≥1.2万
**AI执行规则**: 按人群活跃时间精准发布

### SK-OPR-003: 全赛道小额投放智能测试
**适用场景**: 爆款放大、流量撬动、低成本测款
**核心目标**: 小预算撬动自然流量，ROI≥5
**输入**: 视频数据、人群画像、投放预算
**微步骤**:
1. 单条测试预算≤200元
2. 定向：精准人群，拒绝泛投
3. 优化目标：完播/互动/关注
4. 10分钟监测：完播＜30%直接关停
5. 完播＞50%、互动＞10%智能放量
**输出**: 投放测试与优化报告
**顶级量化判断**: 投放ROI≥5
**AI执行规则**: 只放量优质内容，不停亏钱
