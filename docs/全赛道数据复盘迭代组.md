## 技能列表

### SK-ANA-001: 全赛道数据指标极速诊断
**适用场景**: 单视频/账号复盘、问题定位
**核心目标**: 10秒定位数据短板，准确率100%
**输入**: 播放、完播、点赞、评论、收藏、关注
**微步骤**:
1. 播放低：封面/标题/账号权重
2. 完播低：开头/节奏/内容时长
3. 点赞低：无价值/无情绪共鸣
4. 评论低：无引导/无争议点
5. 关注低：无系列/无期待感
**输出**: 数据诊断与优化方案
**顶级量化判断**: 问题定位准确率100%
**AI执行规则**: 只看数据，不主观猜测

### SK-ANA-002: 全赛道爆款基因提取与固化
**适用场景**: 批量复制爆款、工业化生产
**核心目标**: 偶然爆款变流水线，复用爆款率≥80%
**输入**: 自身&竞品爆款数据、脚本、分镜
**微步骤**:
1. 提取爆款开头钩子类型
2. 提取爆款节奏与切镜频率
3. 提取爆款选题方向
4. 提取爆款封面标题公式
5. 固化为可批量复制模板
**输出**: 爆款基因库+可复制模板
**顶级量化判断**: 模板复用爆款率≥80%
**AI执行规则**: 只提取全赛道通用、可量产基因
