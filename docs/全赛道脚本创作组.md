## 技能列表

### SK-CRE-001: 全赛道3秒黄金钩子撰写
**适用场景**: 所有视频开头留人、提升3秒留存
**核心目标**: 3秒留人率≥75%，全赛道通用
**输入**: 选题、人群痛点、视频核心价值
**微步骤**:
1. 疑问钩子：直击痛点+反问用户
2. 结果钩子：直接给出最终效果/收益/改变
3. 冲突钩子：矛盾/对立/意外场景
4. 反常钩子：违背常识+引发好奇
5. 指令钩子：提醒/警告/命令
6. 数字钩子：强数据+强对比
7. 控制：≤15字，口语化，无废话
**输出**: 5版3秒钩子文案
**顶级量化判断**: 前3秒留存率≥75%
**AI执行规则**: 第一句必须直击情绪或利益

### SK-CRE-002: 全赛道标准化时间轴脚本撰写
**适用场景**: 15s/30s/60s全时长视频拍摄脚本
**核心目标**: 脚本零废稿，拍摄可直接执行
**输入**: 选题、钩子、核心价值、目标时长
**微步骤**:
1. 15s：3s钩子+9s价值+3s互动
2. 30s：3s钩子+20s论证+4s情绪+3s引导
3. 60s：3s钩子+40s干货/故事+12s案例+5s引导
4. 单句台词≤10字，纯口语化
5. 每5秒1个信息点，无空镜无废话
6. 标注重音、停顿、情绪、动作指令
**输出**: 带时间轴的拍摄执行脚本
**顶级量化判断**: 成片率100%，完播率≥60%
**AI执行规则**: 严格按时间轴切割，拒绝模糊描述
