## 技能列表

### SK-STR-001: 全赛道垂直人群5层精准画像拆解
**适用场景**: 全赛道冷启动、账号定位、选题方向
**核心目标**: 输出可直接用于内容创作的精准人群标签
**输入**: 平台人群数据、竞品评论热词、搜索词、热榜关键词
**微步骤**:
1. 提取基础标签：年龄/性别/地域/职业/消费力
2. 提取高频场景：通勤/家务/带娃/上班/学习/熬夜
3. 提取正向需求：期待收益/效果/改变/情绪价值
4. 提取负向痛点：焦虑/麻烦/踩坑/无效/耗时
5. 提取情绪偏好：治愈/爽感/搞笑/焦虑/共鸣/干货
6. 提取决策逻辑：看效果/价格/口碑/便捷性/安全性
7. 输出唯一人群定位标签：人群+场景+痛点+目标
**输出**: 标准化人群画像JSON结构体
**顶级量化判断**: 人群匹配度≥95%，痛点命中率≥90%
**AI执行规则**: 禁止泛化描述，必须精准到具体场景+具体情绪

### SK-STR-002: 全赛道蓝海微赛道挖掘
**适用场景**: 全赛道起号、避内卷、提高爆款率
**核心目标**: 定位低竞争、高爆款、可复制、可变现微赛道
**输入**: 大类关键词、平台近30天爆款池、同赛道账号库
**微步骤**:
1. 三级拆分赛道：大类→中类→小类→微类
2. 统计微赛道万赞爆款视频数量
3. 统计微赛道≥10万粉头部账号数量
4. 计算竞争值=头部账号数/万赞爆款数（≤0.2为蓝海）
5. 验证可复制性：爆款形式≤3种
6. 验证合规性：无医疗/金融/法律/时政强监管
7. 验证变现性：具备商品/服务/商单/引流任一路径
**输出**: 蓝海微赛道清单（竞争值、爆款率、变现方式）
**顶级量化判断**: 连续10条内容爆款率≥40%
**AI执行规则**: 只推荐可复制、低竞争、强变现微赛道

### SK-STR-003: 全赛道7大类爆款选题生成
**适用场景**: 日更选题库、批量内容规划、全品类
**核心目标**: 每条选题自带爆款基因，无三无内容
**输入**: 人群画像、痛点词、平台热词、竞品爆款标题
**微步骤**:
1. 痛点型：人群+场景+痛点+解决方案
2. 干货型：数字+步骤+结果+目标人群
3. 反差型：认知/身份/结果/预期反差
4. 故事型：冲突+反转+情绪+价值输出
5. 热点型：热词+人群+赛道+低风险结合
6. 争议型：安全二选一观点+无违规引导
7. 系列型：主线主题+≥20条可更新子选题
**输出**: 每日50条结构化选题（类型、爆率、风险等级）
**顶级量化判断**: 选题爆款预测准确率≥85%
**AI执行规则**: 禁止无痛点、无情绪、无价值的三无选题
