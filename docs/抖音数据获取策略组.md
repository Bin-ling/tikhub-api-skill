# 抖音数据获取策略组

## 角色
数据获取策略师

## 职责
- 热点追踪与分析
- 竞品数据监测
- 内容效果评估
- 用户画像构建

- 自动化数据采集

## 技能列表

### SK-DATA-001: 抖音热点追踪与分析
**适用场景**: 热点内容创作、快速响应热点
**核心目标**: 实时监控热点变化，发现高潜力内容机会
**输入**: API密钥、监控频率、关注领域
**微步骤**:
1. 调用抖音热搜榜API获取官方热点
2. 调用热点总榜API获取综合热点数据
3. 调用上升热点榜API发现潜力热点
4. 调用搜索热榜API获取用户搜索热点
5. 分析热点趋势和关联度
6. 生成热点内容机会报告
**输出**: 热点监测报告（包含热点趋势、潜力机会、内容建议）
**顶级量化判断**: 热点发现准确率≥95%，响应时间≤15分钟
**AI执行规则**: 实时监控，快速响应，避免过时热点

### SK-DATA-002: 抖音竞品分析
**适用场景**: 竞品策略研究、差异化定位
**核心目标**: 深入分析竞争对手表现，发现差异化机会
**输入**: 目标领域、竞品账号列表、分析维度
**微步骤**:
1. 调用热门账号API发现同领域优质账号
2. 调用账号作品分析API获取竞品表现数据
3. 调用用户信息API获取竞品账号详情
4. 调用用户作品API获取竞品热门内容
5. 分析竞品内容策略和表现指标
6. 生成竞品分析报告
**输出**: 竞品分析报告（包含账号表现、内容策略、差异化机会）
**顶级量化判断**: 竞品覆盖≥90%，分析维度≥5个
**AI执行规则**: 客观分析，避免主观判断，数据说话

### SK-DATA-003: 抖音内容效果评估
**适用场景**: 内容优化、数据驱动决策
**核心目标**: 基于数据反馈持续优化内容创作
**输入**: 视频ID列表、分析周期、评估指标
**微步骤**:
1. 调用视频统计API获取播放、点赞等核心数据
2. 调用视频评论API获取用户反馈
3. 调用评论分析API获取评论词云
4. 调用作品数据趋势API监控数据变化
5. 分析内容表现短板和优势
6. 生成内容优化建议
**输出**: 内容效果评估报告（包含数据指标、用户反馈、优化建议）
**顶级量化判断**: 数据准确率≥99%，优化建议可执行性≥85%
**AI执行规则**: 基于数据说话，避免主观臆断

### SK-DATA-004: 抖音用户画像构建
**适用场景**: 精准内容定位、用户需求挖掘
**核心目标**: 深入了解目标用户群体，优化内容定位
**输入**: 账号ID、分析维度、样本量
**微步骤**:
1. 调用粉丝画像API获取粉丝 demographic 数据
2. 调用作品观众画像API分析视频受众特征
3. 调用粉丝兴趣话题API了解用户兴趣
4. 调用粉丝搜索词API获取用户搜索偏好
5. 整合多维度数据构建完整用户画像
6. 生成用户画像分析报告
**输出**: 用户画像分析报告（包含人口统计、兴趣偏好、行为特征）
**顶级量化判断**: 画像准确率≥90%，维度覆盖≥8个
**AI执行规则**: 数据驱动，精准刻画，避免刻板印象

### SK-DATA-005: 抖音垂直领域数据挖掘
**适用场景**: 细分领域机会发现、专业内容策略
**核心目标**: 深入挖掘特定垂直领域的内容机会
**输入**: 垂直领域ID、分析深度、时间范围
**微步骤**:
1. 调用垂类内容标签API获取领域标签体系
2. 调用垂直领域视频热榜API获取热门内容
3. 调用垂直领域热门账号API发现优质创作者
4. 调用垂直领域话题热榜API了解热门话题
5. 分析垂直领域内容趋势和机会
6. 生成垂直领域机会报告
**输出**: 垂直领域机会报告（包含热门内容、优质账号、机会点）
**顶级量化判断**: 领域覆盖深度≥95%，机会发现准确率≥85%
**AI执行规则**: 专注垂直，深度挖掘，避免泛化分析

### SK-DATA-006: 抖音内容创作辅助
**适用场景**: 内容灵感获取、创作方向指导
**核心目标**: 为内容创作提供数据支持和灵感
**输入**: 内容类型、目标领域、创作需求
**微步骤**:
1. 调用低粉爆款榜API发现潜力内容形式
2. 调用高完播率榜API学习内容结构
3. 调用高点赞率榜API分析用户偏好
4. 调用高涨粉率榜API了解涨粉策略
5. 提取可复制的内容元素和结构
6. 生成创作灵感和模板
**输出**: 创作辅助报告（包含灵感来源、结构模板、优化建议）
**顶级量化判断**: 灵感实用性≥90%，模板可执行性≥85%
**AI执行规则**: 数据驱动灵感，避免主观臆断
